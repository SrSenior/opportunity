# Opportunity (Puzzle Clicker Game in Unity - C#)

## 📌 Descripción

Juego 2D desarrollado en Unity donde el jugador debe resolver una serie de decisiones lógicas entre dos opciones (puertas), basándose en pistas generadas de forma dinámica.
El objetivo es seleccionar la opción correcta a partir de información obtenida con las herramientas que tenemos dentro del juego.

## 🎮 Demo jugable

Puedes probar la última versión del juego aquí:
👉 [Jugar en Itch.io](https://sr-senior.itch.io/opportunity-the-phone-man)

## ⚙️ Tecnologías

* Unity (2D)
* C#
* Sistema de eventos de Unity
* Generación de lógica aleatoria

## 🧠 Mecánicas principales

* Sistema de selección entre dos opciones (puertas)
* Generación aleatoria de características en cada partida
* Lógica de validación para determinar la opción correcta
* Dificultad implícita basada en combinaciones posibles

## 🔁 Sistema de aleatorización

Cada partida presenta variaciones en:

* Propiedades de las puertas
* Condiciones para determinar la respuesta correcta
* Combinaciones de reglas

Esto permite rejugabilidad y evita soluciones predecibles.
